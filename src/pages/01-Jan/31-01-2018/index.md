# Arrow functions 



```js
//before
function foo() {
    return 2;
}
 //ES6
foo = x => 2;
```


# Ternary operator

```js
//before
function foo(x) {
    if(x > 5) return x;
    else return 1;
}

//ES6
var foo = x => x > 5 ? x : 1;
```
